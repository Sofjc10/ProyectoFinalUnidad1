# EduConnect – Prototipo Web

## Descripción

EduConnect permite a los padres de familia seguir el comportamiento escolar de sus hijos: asistencia, uniforme y salidas pedagógicas. Este prototipo muestra la interfaz básica de la plataforma.

## Funcionalidades implementadas

- Página de inicio con:
  - Carousel de imágenes.
  - Tres secciones: Asistencia, Uniforme y Salidas Pedagógicas.
- Página 2 con:
  - Formulario para registrar asistencia.
  - Tabla para mostrar historial de asistencia.

## Requisitos No Funcionales (planificados)

- Fácil de usar.
- Responsive en celulares, tablets y computadores.
- Seguridad y acceso mediante usuario (planeado para versiones futuras).

## Cómo ejecutar

1. Abrir la carpeta del proyecto en Visual Studio Code.
2. Abrir `index.html` o `pagina.html` en un navegador.
3. Navegar entre la página de inicio y la página 2 usando la barra de navegación.
